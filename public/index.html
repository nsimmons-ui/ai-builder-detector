How Detection Works

Every URL is scored across two layers. Platform detection runs first and takes priority. If no platform is found, AI heuristics look for code patterns left by AI coding assistants.

Scoring

Signals are weighted by confidence and grouped by category (each category capped at 15 pts):

HIGH = 10 pts MEDIUM = 5 pts LOW = 2 pts

Platform threshold: ≥ 5 pts  ·  AI heuristic threshold: ≥ 10 pts

Output Buckets

Platform-assisted — built with a no-code or AI site builder (Framer, Wix, Shopify, etc.)
AI-assisted — code patterns consistent with an AI coding assistant (Claude, Cursor, Copilot, etc.)
No AI signals — no identifiable signals found; likely hand-written
Unknown — could not fetch or analyse the page

Platform Signals

Framer
  • Subdomain .framer.website / .app / .ai
  • Server header starting with Framer/
  • Generator meta tag
  • Data attributes data-framer-*
  • CSS variables --framer-font-family etc.
  • CDN framerusercontent.com
Webflow
  • Subdomain .webflow.io
  • Generator meta tag
  • HTML comment "This site was created in Webflow"
  • Data attributes data-wf-*
  • CSS classes w-nav, w-container, w-row
  • JS globals window.Webflow
  • CDN assets.website-files.com
Wix
  • Subdomain .wix.com
  • Server header: Pepyaka
  • Header x-meta-site-is-wix-site: 1
  • Generator meta tag
  • CSS variables --color_0, --wix-ads-height
  • DOM IDs SITE_CONTAINER, masterPage
  • CDN static.parastorage.com
  • Data attributes data-testid="comp-*", data-mesh-id
Squarespace
  • Subdomain .squarespace.com
  • Server header containing "Squarespace"
  • Generator meta tag
  • JS globals SQUARESPACE_ROLLUPS, Y.Squarespace
  • CSS classes sqs-block, sqs-layout
  • CDN static*.squarespace.com
Shopify
  • Subdomain .myshopify.com
  • Header powered-by: Shopify (survives headless)
  • Header x-shopid (survives headless)
  • Header shopify-complexity-score
  • Cookie _shopify_* (survives headless)
  • JS globals window.Shopify, ShopifyAnalytics
  • CDN cdn.shopify.com
Lovable
  • Subdomain .lovable.app
  • Generator meta tag
  • JS globals window.__lovable*
  • Asset paths /lovable-uploads/[UUID]
Bolt
  • Subdomain .bolt.new, .stackblitz.io
  • Server header containing "bolt"
  • Header server-version
  • Generator meta tag
  • JS globals window.__bolt*
v0 (Vercel)
  • Subdomain .v0.dev
  • Generator meta tag
  • JS globals window.__v0*
GitHub Pages
  • Subdomain .github.io
  • Server header GitHub.com
  • HTML comment mentioning "Copilot Workspace" (low)
Cursor AI
  • HTML comment mentioning "Cursor AI" (low)
  • Meta tag with content "Cursor AI" (low)

AI Heuristic Signals

These fire only when no platform is detected. Score ≥ 10 triggers AI-assisted.

shadcn/ui patterns HIGH 10 Specific class combos from shadcn components (Card, Button, Badge, Dialog), @radix-ui/ imports, cn()/twMerge
Inter font HIGH 10 fonts.googleapis.com/css.*Inter or font-family:.*Inter
Tailwind stack MED 5 ≥2 of: verbose flex/grid combos, sm:/dark: prefix stacking, color scale shades (slate-500 etc.)
Vite + React SPA MED 5 /assets/index-[hash].js (Vite output) + empty <div id="root">
Lucide icons MED 5 lucide-react in script src or lucide reference in HTML/JS
Vercel / Netlify hosting MED 5 Response headers x-vercel-id or x-nf-request-id
AI generator meta tag HIGH 10 <meta name="generator"> containing v0, bolt, lovable, cursor, copilot, claude, chatgpt, gpt
Prototype hosting LOW 2 Subdomain on vercel.app, netlify.app, pages.dev, railway.app, render.com, fly.dev
Over-commenter comments LOW 2 Tutorial-style comments explaining obvious code ("This function", "Here we", "Hero section", etc.)
Generic numbered naming MED 5 ≥5 occurrences of .container-1, .wrapper-2, .section-3 style names
Placeholder / hallucinated links LOW 2 href="#", href="https://example.com", /placeholder, lorem ipsum in links
AI buzzword density HIGH 10 / 5 / 2 Visible page text stripped of HTML and scanned for LLM marketing words (seamless, robust, elevate, revolutionize, cutting-edge, empower, leverage, etc.) — high density = likely AI-written copy
"Too clean" code LOW 2 Zero developer comments (no // old, TODO, FIXME) combined with high className= density (>0.15 ratio) — AI code lacks the messy traces of human development
Uniform list structure MED 5 / 2 5+ <li> items with <15% length variation (coefficient of variation) — AI generates suspiciously parallel, equal-length copy
